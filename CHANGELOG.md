# CHANGELOG

### v1.0.1

1. 修复部署在设备上直接访问时出现的致命错误：  
   ```
   Fatal error: Uncaught Error: Failed opening required '/storage/emulated/0/htdocs/config.php' (include_path='.:/usr/local/lib/php') in /storage/emulated/0/htdocs/includes/common.php:19
   Stack trace:
   #0 /storage/emulated/0/htdocs/index.php(3): require_once()
   #1 {main} thrown in /storage/emulated/0/htdocs/includes/common.php on line 19
   ```
   已修复该问题：改进了对配置文件路径的检测与兼容性处理，在缺失或不可读时给出友好提示并避免抛出致命错误，提升在移动设备/特殊部署环境下的稳定性。

### v1.0.0

1. 全新界面与响应式设计：重构桌面与移动端界面，优化文件预览页面与小图标展示，提高用户视觉与交互体验。
2. 媒体播放与审核：视频播放器切换为 ckplayer，音乐播放器使用 APlayer，支持视频文件的人工审核与可拖拽播放（断点续传/跳转）。
3. 多云存储支持与直连能力：新增阿里云 OSS（含图片违规检测）、七牛、腾讯云 COS、又拍云、华为 OBS 等云存储接入，并支持直接对接云存储 API 上传与直链下载（无需本机中转，提升上传/下载速度）。
4. 上传可靠性与性能增强：默认使用分块上传，上传前计算文件 hash 支持极速秒传与完整性校验，新增上传 API 接口以便第三方或脚本调用。
5. 下载体验改进：支持断点续传、直链下载模式并修复中文文件名乱码问题，显著提升下载稳定性与兼容性。
6. 用户系统与记录：新增用户系统，登录用户可保留上传记录并可在后台查看与管理上传历史。
7. 搜索与后台管理改进：新增文件搜索功能并修复后台文件搜索相关问题；后台支持批量封禁/解封用户及更多管理操作；所有站点设置均可在后台修改。
8. 部署与兼容性修复：修复 SAE 兼容性问题、二维码显示问题及若干部署/环境相关错误，提升在不同平台的可用性。
9. 安全修复与加固：修复高危漏洞并进行多项安全修补和稳定性优化。
10. 其他修复与优化：包括性能优化、页面样式调整与若干已知问题修复，提升整体稳定性与用户体验。
